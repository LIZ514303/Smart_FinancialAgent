# AI Agent 智能理财助手 (Smart Finance Agent)

本项目是一个基于 Python 构建的ai智能理财 Web 应用。它集成了大型语言模型（LLM）作为 AI Agent，帮助用户自动解析账单、进行收支统计，并提供个性化的理财规划建议。

## 🌟 核心功能

1. **账单自动解析**：支持上传 CSV 格式的交易账单（如支付宝/微信账单导出格式）。
2. **收支可视化**：自动生成多维度的消费统计图表（饼图、柱状图）。
3. **AI 理财规划师**：基于用户的真实消费数据，AI Agent 将深度分析消费习惯，并给出下个月的预算建议和理财规划。
4. **每周目标与每日提醒**：根据生成的理财规划以及用户的个人调整，制定出每周理财目标与每日的任务并提醒用户。
